## Design Elements

### Color Palette
- **Primary Background**: #13191D (Dark blue-gray)
- **Secondary Background**: #1a2025 (Lighter blue-gray)
- **Accent Color**: #f2f827 (Bright yellow)
- **Text Primary**: #ffffff (White)
- **Text Secondary**: #a8b0b7 (Light gray)
- **Borders**: #34383b (Medium gray)

### Typography
- **Headers**: Monospace font, bold
- **Body Text**: Sans-serif, regular weight
- **Accent Text**: Yellow color (#f2f827)

### UI Elements
- **Cards**: Rounded corners (12px), subtle borders
- **Buttons**: Rounded corners (8px), yellow accent
- **Icons**: Consistent style, yellow accent color
- **Charts**: Dark theme with yellow highlights

## Implementation Notes

1. **Fallback Handling**: All images include fallback content that displays when images fail to load
2. **Responsive Design**: Images should work well on different screen sizes
3. **Loading States**: Consider adding loading animations for better UX
4. **Accessibility**: Include proper alt text for all images
5. **Performance**: Optimize images for web delivery (compress appropriately)

## File Naming Convention
- Use kebab-case for file names
- Include the solution name as prefix
- Use descriptive names that indicate the content
- Example: `brm-intelligence-dashboard.png`
